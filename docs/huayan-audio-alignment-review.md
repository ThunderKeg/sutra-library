# 《华严经》第一册音频对齐审校记录

## 结论

- 第一册原书第 2–615 页共 614 页均有独立对齐记录；其中 554 页有可读诵文本，60 页为卷末影印附页并标记为不读诵、零时长。
- 10 条音轨页间边界全部单调且连续，最大页间时间差为 0 秒；所有时间点均在各自音轨时长内。
- 逐页匹配覆盖率中位数为 92.50%；各轨精确字符覆盖率最低 62.23%，加入全局同音字锚点后各轨最低覆盖率为 87.84%。
- 卷七的两个网页章节共用 `1018.m4a`，原书第 380 页在 927.400 秒结束，第 381 页从 927.400 秒开始，跨章节无断点。
- 正文文件清单哈希复算仍为 `8e566e27ce88008eb7a27a5d27b615c69377ea23`，且正文索引和章节 JSON 相对任务开始时的 `HEAD` 无差异。

## 方法复核

1. 粗估阶段按音轨实测时长与各页有效汉字数分配初始边界。
2. 精对齐阶段使用本地 `faster-whisper small` 的词级时间戳，把识别字符与现有校本文字做全局编辑距离匹配，再用同音字匹配补充单调锚点。
3. `1012.m4a` 自带未收入本对齐范围的开经偈，正文“如是我闻”在 127.160 秒被单独核定为原书第 2 页起点。
4. `1013.m4a`–`1021.m4a` 的首个网页页本身含呼钟声偈与开经偈，因此首页面从音轨 0 秒开始，下一页边界再由正文锚点确定。
5. 独立检查器 `scripts/check_huayan_audio_alignment.py` 重新读取正文页身份，检查 614 页完整性、音轨映射、时间范围、单调连续性、卷七跨章节边界和正文未改动状态。

## 低置信度页复核

共有 12 个文本页的单页覆盖率低于 50%。原书第 49、111、173、239、295、365、437、497、565 页都是卷首呼钟声偈／开经偈；唱诵识别率低，但其起点固定为音轨 0 秒，终点由下一页正文锚点限定。原书第 430、559、608 页是相应音轨最后的文本页，其后均为不读诵的影印附页；终点由音轨末段锚点限定。其余正文页无连续低覆盖区。

审校未使用识别结果修改任何经文；识别文本和本地音频仅作为时间定位中间材料，均不提交到仓库。
